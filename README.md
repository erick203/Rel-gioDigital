# 📝 Descrição

Este é um projeto de um **Relógio Digital** moderno e elegante, desenvolvido com foco em estilização avançada e manipulação do DOM em tempo real. A interface conta com um design *Premium* utilizando o efeito **Glassmorphism** (imitação de vidro fosco), iluminação neon sutil (glow effect) e animações fluidas de interatividade ao passar o mouse.

---

## 🚀 Recursos Premium

* **Efeito Glassmorphism:** Cards com fundo semi-transparente e desfoque de fundo (`backdrop-filter`).
* **Design Responsivo:** Se adapta perfeitamente a diferentes tamanhos de tela.
* **Efeito Glow:** Iluminação em estilo neon azul nos números para melhor leitura e estética futurista.
* **Micro-interações:** Animações suaves de elevação (`hover`) ao passar o ponteiro do mouse sobre os blocos de tempo.
* **Sincronização em Tempo Real:** Atualização precisa a cada segundo utilizando a API de data do JavaScript.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero utilizando as tecnologias web fundamentais:

* **HTML5:** Estruturação semântica dos blocos de tempo.
* **CSS3:** Estilização avançada, gradientes dinâmicos, transições e efeitos visuais modernos.
* **JavaScript (ES6):** Lógica para capturar as horas do sistema e atualizar a interface em tempo real.
* **Google Fonts:** Utilização da fonte *Poppins* para um visual limpo e profissional.

---

## ⚙️ Como Executar o Projeto

1. Faça o clone ou baixe os arquivos deste repositório.
2. Certifique-se de que os arquivos `index.html`, `styles.css` e `script.js` estão na mesma pasta.
3. Abra o arquivo `index.html` diretamente em qualquer navegador web (Chrome, Edge, Firefox, Safari).
